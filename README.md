## Guess The Capital Game

This is a small web-based game where users try to guess the capital cities of different countries.

I worked on this project while learning cloud computing and deployment basics. The main goal of this project was not just building the game, but understanding how applications can be hosted and deployed on the cloud.

## Live Demo

https://guess-the-capital.2af7sxxl64l6.us-south.codeengine.appdomain.cloud/

##  Deployment

The application was deployed using IBM Cloud Code Engine through a guided cloud lab environment.

## Tech Used

* HTML
* CSS
* JavaScript
* IBM Cloud Code Engine

## What I Learned

Through this project, I got introduced to:

* cloud deployment workflows
* hosting applications online
* working with cloud environments
* basic application deployment concepts

This project also helped me understand how cloud platforms can be used to run web applications in real-world environments.
